# Wireless Network Performance Troubleshooting

## Resolution steps

1. Run a connection speed test while connected to the affected Wi-Fi
   network and record the download/upload throughput.
2. Restart the wireless adapter (toggle Wi-Fi off/on, or toggle Airplane
   Mode on laptops).
3. Move closer to the access point — signal attenuation from distance or
   physical obstruction is the most common root cause of reported "slow"
   performance.
4. If working remotely, assess concurrent network load (streaming
   devices, gaming consoles, smart-home devices competing for
   bandwidth).
5. Forget and reconnect to the network: **Settings → Network → Wi-Fi →
   Forget This Network**, then reconnect and re-authenticate.
6. Power-cycle the router (30-second minimum power-off) if managing a
   personal home network.

## When to contact the Service Desk instead

Escalate if multiple co-located users report the same degradation
(indicative of an access point issue requiring Facilities/Network team
escalation), or if degraded performance persists on a wired connection
after completing the steps above.
