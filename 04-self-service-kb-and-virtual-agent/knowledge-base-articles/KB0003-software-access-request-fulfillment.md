# Software Access Request Fulfillment

## Resolution steps

1. Navigate to the Self-Service Portal and select **"Request Something"**.
2. Select the applicable catalog item: Software Installation, System
   Access, or Shared Drive / Folder Access.
3. Provide the requested item and a business justification (a single
   sentence is sufficient — e.g. "Visio required for architecture
   diagrams on Project X"), and confirm the assigned manager if not
   auto-populated.
4. Submit. An automated approval request routes to the requester's
   manager.
5. Upon approval, the request routes automatically to the fulfillment
   queue; a notification confirms completion.

## Expected fulfillment timelines

| Request type | Typical turnaround |
|---|---|
| Standard catalog software | Same business day following approval |
| Non-catalog / custom software | 2–3 business days (requires security review) |
| System or database access | 1–2 business days following manager and data-owner approval |

## When to contact the Service Desk instead

Escalate if manager approval has been pending more than 2 business days,
or if the request is time-critical and blocking active work — reference
the request number when contacting the Service Desk to expedite review
or identify a stalled approval step.
