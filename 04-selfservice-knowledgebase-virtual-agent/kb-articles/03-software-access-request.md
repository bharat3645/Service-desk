# Requesting New Software or System Access

**Category:** Access Management
**Audience:** All employees
**Estimated time:** 2 minutes to submit; approval time varies

## Steps

1. Go to the Self-Service Portal and select **"Request Something"**.
2. Choose the relevant catalog item: Software Installation, System Access,
   or Shared Drive/Folder Access.
3. Fill in: what you need, the business justification (one sentence is
   fine — e.g. "Need Visio for architecture diagrams on Project X"), and
   your manager's name if not auto-populated.
4. Submit. Your manager receives an automatic approval request.
5. Once approved, the request routes automatically to the fulfillment
   queue — you'll get a notification when it's completed.

## Typical turnaround

- Standard software from the approved catalog: same business day after approval
- Non-catalog / custom software: 2-3 business days (requires security review)
- System/database access: 1-2 business days after manager + data owner approval

## When to contact the Service Desk instead

If your request has been pending manager approval for more than 2
business days, or if this is urgent and blocking your work — call in and
reference your request number so we can expedite or check for a stuck
approval step.
